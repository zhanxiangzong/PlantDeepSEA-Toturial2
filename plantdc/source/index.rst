`PlantDeepSEA <http://plantdeepsea.ncpgr.cn/>`_ Documentation
========================================
Welcome! This is the documentation for PlantDeepSEA, a deep learning web service based on Selene Python SDK and Django. The web sever address is located `here <http://plantdeepsea.ncpgr.cn/>`_. PlantDeepSEA is freely accessible for all users.

**Citation** : Hu Zhao#, Zhuo Tu#, Yinmeng Liu, Zhanxiang Zong, Jiacheng Li, Hao Liu, Feng Xiong, Jinling Zhan, Xuehai Hu, and Weibo Xie* (2021). PlantDeepSEA, a deep learning-based web service to predict the regulatory effects of genomic variants in plants. *Nucleic Acids Research*, doi: 10.1093/nar/gkab383
